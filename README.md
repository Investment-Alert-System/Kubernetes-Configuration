# Installation 👨‍🔧
## MacOS 💻
```shell
brew install terraform
brew install --cask docker
```
## Windows 🪟
```powershell
choco install terraform
choco install docker-desktop
```
# Apply Terraform 🌎
```shell
terraform init
terraform plan
terraform apply
```
